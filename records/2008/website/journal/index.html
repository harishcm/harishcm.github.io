      1. Linking to the Resources section within the main Menu

          I should make the Resources section easy to navigate, and link to it in the main Menu. This is one of the most important sections. It would also partially address the issue of adding links to other websites.

      2. Merging the Records section into the Sidebar

          I should also include in as many links to the Records section as possible in the Sidebar. Perhaps, using only the map plugin on the sidebar page will prevent excessive full wiki rebuilds.     4. Possible Further Changes to Website
      1. Making space in the layout itself for the most important messages

          Another idea would be to make space for including the most important messages in the layout itself. One possible way would be making space for a subtitle. Another way would be to make space for quotes and statements, whether mine or others’.

      2. Adding a comments box for the entire website

          I am still not yet ready to add comments for each and every page on the website. The main reason is that I will then have to deal with dynamic or semi-dynamic content, which presents a whole new level of challenges. I will also have to deal with issues such as spam.

          Nevertheless, the benefits of having users being able to post to the website can the tremendous. It could help greatly in receiving feedback about the website and its contents. It could possibly help in establishing cooperative learning tie-ups.

          A start could be to add a single comments box for the entire website. This could be within the sidebar, or possibly on a page by itself.
